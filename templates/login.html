Helpdesk Micro is a small, lightweight call logging system designed to work for small internal IT teams.

What can it do?

With full CRUD functionality, users can add tickets, end users and comment each ticket individually to keep each call up to date for the team.

New member to the IT team?

An exsting Admin user can add that member to the team with a secure hashed password. Once logged in the new user can reset their password.
